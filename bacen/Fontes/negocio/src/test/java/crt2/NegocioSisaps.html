Acúmulo de perfis de consulta

  • Para todos os perfis de risco de consulta, a opção do menu sempre deverá ser 'Consulta'.
  • A aba de Consulta deve ser única, isto é, se o administrador conceder dois ou mais perfis de consulta para o mesmo usuário, o menu Consulta deverá abrir a tela de consulta do perfil mais abrangente.
  • Ordem de abrangência da consulta (da mais abrangente para a menos abrangente):
    • Consulta tudo.
    • Consulta somente não bloqueados.
    • Consulta somente resumo de perfil não bloqueado.
