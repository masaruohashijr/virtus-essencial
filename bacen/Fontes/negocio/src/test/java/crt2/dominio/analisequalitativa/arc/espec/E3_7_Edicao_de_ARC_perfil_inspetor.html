E3.7. Edição de ARC perfil inspetor

Pré-requisitos

Deve existir ARCs designados nos estados 2 - Designado ou 3 - Em edição para o inspetor logado no sistema.

Perfil de usuário

Inspetor.

Item de menu

Tela inicial da aplicação para o perfil inspetor

Telas

Título: Edição de ARC

Trilha: Não se aplica

Código de tela: [APSFW0604]

    Comandos:
  • Salvar nota do ARC e justificativa da nota do ARC - Para o ARC.
  • Salvar nota do elemento - Para cada elemento do ARC.
  • Salvar justificativa associada ao item a avaliar do elemento - Para cada item a avaliar do elemento.
  • Anexar arquivo na justificativa da nota do elemento - Para cada item a avaliar do elemento.
  • Visualizar arquivo anexado na justificativa da nota do elemento - Para cada item a avaliar do elemento.
  • Excluir arquivo anexado na justificativa da nota do elemento - Para cada item a avaliar do elemento.
  • Salvar tendência do ARC e justificativa da tendência do ARC - Para o ARC.
  • Concluir e encaminhar para a análise do Supervisor - Para o ARC.

Descrição

Funcionalidade básica de edição do ARC designado para o inspetor.

Funcionalidades disponíveis apenas para o perfil Inspetor.

Regras específicas

Detalhamento:
  • Tela de edição do ARC para o inspetor atualizador.
  • Trantando-se da primeira edição do ARC para a ES, os campos abaixo não possuem valor para exibição:
    • Nota vigente do ARC.
    • Nota vigente do elemento - Para cada elemento.
    • Justificativa vigente associada ao item a avaliar do elemento - Para cada item a avaliar de cada elemento.
    • Anexos vigentes associados ao item a avaliar do elemento - Para cada item a avaliar de cada elemento.
    • Tendência vigente - Para o ARC.
  • Caso contrário, sempre serão exibidas as informações vigentes e uma área de edição para informar os novos valores das informações (nota, justificativa, etc)
  • Cada ARC possui elementos associados.
  • Cada elemento possui itens a avaliar associados.
  • Elemento é parâmetro.
  • Item a avaliar do elemento é parâmetro.
  • Tendência é parâmetro.
  • Nota é parâmetro.
  • A aplicação deve gerar alertas indicando a existência de informações que foram editadas, mas que não foram salvas pelo usuário.
    • "Ná informações não salvas nesta página." - Trata-se de um alerta geral abaixo do título da página.
    • Atenção texto não salvo - Trata-se de um alerta específico para um campo texto e deve ser exibido próximo ao campo.
    • Atenção informações não salvas - Trata-se de um alerta específico para um conjunto de informações e deve ser exibido próximo ao conjunto.
    • Atenção nota não salva - Trata-se de um alerta específico para um campo nota e deve ser exibido próximo ao campo.
  • Ao salvar a nota de cada elemento, o campo Nova nota do ARC deve ser 'recalculado' sendo atribuido a ele a menor das notas dadas aos elementos do ARC.
  • Ciclo de vida do ARC:
    • 1 - Previsto: na criação do ARC após liberação da matriz na inclusão manual do ciclo ou na inclusão automática do ciclo, ambos por usuário com perfil supervisor (responsável da ES).
    • 2 - Designado: após a designação de atualização do ARC pelo usuário com perfil supervisor (responsável da ES).
    • 3 - Em edição: após alteração do ARC pelo usuário designado. Permanece neste estado até o usuário designado concluir o preenchimento.
    • 4 - Preenchido: usuário informa que finalizou a inclusão de notas.
    • 5 - Análise delegada: supervisor responsável da ES delega a análise do ARC para outro usuário.
    • 6 - Em análise: supervisor responsável da ES ou outro usuário por ele delegado está analisando o ARC.
  • Ao se salvar pela primeira vez qualquer seção da tela o estado do ARC deve sair de Designado para Em edição.
  • Ao se clicar em Concluir e enviar para análise do Supervisor o estado do ARC deve sair de Em Edição para Preenchido.

Cenários de teste

Observação: agrupar a maior quantidade de itens de teste no mesmo teste.

Negócio.

  • C001 Falha - Inspetor altera e salva ARC não designado para ele. Mensagem:
    • "Inspetor não designado para o ARC."
  • C002 Falha - Inspetor altera e salva ARC com estado diferente de 2 - Designado ou 3 - Em edição. Mensagem:
    • "ARC no estado estado_do_ARC não pode ser alterado pelo Inspetor."
  • C003 Falha - Ciclo e matriz em estados que não permitem edição de ARC. Mensagem:
    • Não é possível edição de ARC em ciclo no estado diferente de Em andamento.
    • Não é possível edição de ARC de matriz no estado diferente de Vigente.
  • C004 Sucesso - Ao se clicar no comando Salvar da seção que representa o elemento. Mensagem:
    • Nota para nome_do_elemento salva com sucesso.
  • C005 Sucesso - Ao se clicar no comando Salvar da seção que representa o item de avaliação do elemento. Mensagem:
    • Justificativa para nome_do_item_de_avaliação salva com sucesso.
  • C006 Sucesso - Salvando a nota de um elemento, o campo Nova nota do ARC deve ser igual a menor das notas dadas aos elementos do ARC.
  • C007 Falha - Selecionar uma nota para Nova nova do ARC (ajustada) sem preencher a respectiva justificatica e salvar. Mensagem:
    • A justificativa para Nova nova do ARC (ajustada) é de preenchimento obrigatório.
  • C008 Falha - Selecionar uma tendência sem preencher a respectiva justificatica e clicar no salvar. Mensagem:
    • A justificativa para tendência é de preenchimento obrigatório.
  • C009 Falha - Não selecionar um valor para tendência e clicar em Concluir e enviar para análise do Supervisor. Mensagem:
    • O campo tendência é de preenchimento obrigatório.
  • C010 Falha - Clicar em Concluir e enviar para análise do Supervisor sem informar notas para todos os elementos do ARC. Mensagem:
    • O campo Nova nota do elemento de nome_do_elemento é de preenchimento obrigatório.
  • C011 Falha - Clicar em Concluir e enviar para análise do Supervisor sem informar, ao menos, uma justificativa para os itens de avaliação de cada elemento quando a nota do elemento for diferente de N/A. Mensagem:
    • É necessário informar, ao menos, uma justificativa para os itens de avaliação dos elemento do ARC.
  • C012 Falha - Clicar em Concluir e enviar para análise do Supervisor informando, ao menos, uma justificativa para os itens de avaliação do elemento quando a nota deste for N/A. Mensagem:
    • Não é permitida a justificativa de item de avaliação quando a nota do elemento é igual a N/A.
  • C013 Sucesso - Ao se salvar pela primeira vez qualquer seção da tela o estado do ARC deve sair de Designado para Em edição.
  • C014 Sucesso - Clicar em Concluir e enviar para análise do Supervisor. Mensagem:
    • ARC concluído com sucesso.
  • C015 Sucesso - ao se clicar em Concluir e enviar para análise do Supervisor o estado do ARC deve sair de Em Edição para Preenchido.
