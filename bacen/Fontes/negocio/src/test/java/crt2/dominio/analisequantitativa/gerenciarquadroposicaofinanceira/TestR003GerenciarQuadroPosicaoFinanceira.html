Cenário
[CT001] - Selecionar uma data-base sem quadro vigente e sem novo quadro salvo previamente.
Dado que
  • Os dados encontram-se aqui.
Ação pretendida
  • Após o usuário logado selecionar o perfil risco '2'; e
  • Entrar na tela para gerenciar o quadro da posição financeira selecionando a data-base '201401'; e
  • Salvar todos os dados.
Resultado esperado
  • Receber a mensagem:
    • A seleção de contas de ativo foi salva com sucesso.
    • A seleção de contas de passivo foi salva com sucesso.
    • Os ajustes de patrimônios de referência foram salvos com sucesso.
    • Os ajustes de resultados foram salvos com sucesso.
    • Os ajustes de índices foram salvos com sucesso.
  • O novo quadro foi salvo.
