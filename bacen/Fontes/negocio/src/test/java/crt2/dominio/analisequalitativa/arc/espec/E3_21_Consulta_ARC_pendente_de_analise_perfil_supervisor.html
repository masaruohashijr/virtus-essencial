E3.21. Painel do Supervisor

Pré-requisitos

Deve existir um ciclo SRC no estado em andamento cadastrado no sistema de acordo com a metodologia da ES

Uma matriz no estado 2 - Vigente

ARCs nos estados Preenchido, Análise delegada e Em análise de ciclo que o supervisor seja titular ou ARC para o qual ele tenha sido delegado como analisador

Perfil de usuário

Supervisor.

Item de menu

Tela inicial da aplicação para o perfil supervisor

Telas

Título: Painel do Supervisor

Início -- Painel do Supervisor

Código de tela: [APSFW0000]

Comando: Detalhar

Descrição

Funcionalidade disponível apenas para o perfil Supervisor.

Funcionalidade básica de listar os ARCs delegados ou sob responsabilidade do supervisor.

Regras específicas

Detalhamento:
  • A funcionalidade exibe os ARCs de ciclo do qual o supervisor seja responsável ou ainda os ARCs para os quais o supervisor tenha sido delegado. Estado válidos:
    • Preenchido
    • Análise delegada
    • Em análise
  • A funcionalidade deve exibir, conforme protótipo, os elementos de tela.
  • O botão Detalhar abre a tela de detalhamento para o ARC selecionado.

Cenários de teste - Negócio.

Observação: agrupar a maior quantidade de itens de teste no mesmo teste.

  • C001 Falha - Clicar no botão Detalhar sem selecionar um ARC. Mensagem:
    • "Seleção de ARC é obrigatória."
  • C002 Falha - Consulta não retorna dados (lista de ARCs vazia). Mensagem:
    • "Não há ARCs pendentes de análise."
  • C003 Sucesso - Consulta retorna dados de acordo com tabelas dado_que

Cenários de teste - Web.

  • Teste web equivalente - C001 - Conferir apenas a mesangem.
  • Teste web equivalente - C002 - Conferir apenas a mesangem.
  • Conferir - se a listagem de ARCs obedece às regras dispostas acima.
  • Sucesso - Testar a navegabilidade dos comandos Detalhar e Voltar.
