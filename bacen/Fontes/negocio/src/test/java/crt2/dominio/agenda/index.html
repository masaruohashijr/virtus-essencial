  • Observar as regras gerais definidas em meus comitês.

Cenários de testes de negócio
Assinar ata
  • Utilizar o protótipo clicando no botão de alterar da aba 'Comitês com assinatura de ata pendente'.
  • Observar as regras definidas em listar comitês com assinatura de ata pendente.
  • Regras gerais:
    • Caso o usuário clique no botão salvar sem marcar o checkbox, o sistema exibirá a seguinte mensagem de erro: "A ata não foi assinada."
    • Ao clicar no link da ata:
      • Utilizando o parâmetro 'Ata corec', gerar PDF para o download do arquivo.
    • Ao salvar:
      • O sistema deverá voltar para a tela da agenda, atualiando a lista de lista de comitês com assinatura de ata pendente e a lista de comitês com ata assinada.
Cenários de teste de negócio

Listar comitês com ata assinada

  • Observar as regras gerais definidas em meus comitês.
  • Deverão ser listados os comitês que o usuário logado participa e que ele assinou a ata.
Cenários de testes de negócio
Detalhar ata assinada
  • Utilizar o protótipo clicando no botão de detalhar da aba 'Comitês com ata assinada'.
  • Observar as regras gerais definidas em listar comitês com ata assinada.
  • Exibição da ata:
    • Caso exista uma ata do COREC anexada ao sistema:
      • Exibir link com o nome do arquivo para download.
    • Senão:
      • Exibir link conforme o protótipo e ao clicar no link da ata utilizando o parâmetro 'Ata corec', gerar PDF para o download.
