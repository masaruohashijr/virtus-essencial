Iniciar ciclo SRC

  • Utilizar o protótipo.
  • Seguir orientações descritas em APSFW0201_Painel_do_supervisor.xlsx.
  • Criar ANEFs:
    • Para cada ANEF cadastrado na metodologia vigente:
      • Criar dois ANEFs sendo um rascunho e outro vinculado ao perfil de risco (vigente) para o novo ciclo que está sendo criado com o status 'Previsto'.
      • Criar os percentuais para os ANEFs do passo anterior (um rascunho e outro vigente) com base no valor default da metodologia.
