E5.11. Consulta ciclo ativo da ES perfil supervisor - Iteração 02

Pré-requisitos

Deve existir um ciclo ativo para ES sob a responsabilidade do supervisor logado no sistema.

Perfil de usuário

Supervisor.

Item de menu

Tela inicial da aplicação para o perfil supervisor

Ciclo SRC -- Consulta

Telas

Título: Relação de ciclos SRC ativos

Trilha: Ciclo SRC -- Consulta

Código de tela: [RATFW0302]

Comandos (Ações): Perfil de risco (equivale ao detalhar ou selecionar), Análise quantitativa, Avaliação de riscos e controles - ARC, Edição da matriz de riscos e controles (poderão surgir outros comandos).

Comandos (Tela): Voltar (somente quando a tela for exibida através do menu, este comando não é exibido na tela inicial da aplicação).

Descrição

Funcionalidade básica de listar os ciclos SRC ativos.

Funcionalidades disponíveis apenas para o perfil Supervisor.

Regras específicas

Detalhamento:
  • Tela inicial da aplicação para o supervisor.
  • Tela pode ser exibida do mesmo modo através de opção de menu para o supervisor.
  • A inclusão é feita pelo usuário supervisor através do menu Administração -- Ciclo SRC -- Inclusão.
  • A tela inicial da aplicação deve exibir a relação dos ciclos ativos que estejam sob a responsabilidade do supervisor logado no sistema. Ciclos nos estados:
    • Em andamento
    • Corec
  • Esta tela exibe uma coluna de ações que permite acionar para cada ES as telas:
    • Perfil de risco
    • Análise quantitativa
    • Avaliação de riscos e controles - ARC
    • Edição da matriz de riscos e controles.
  • Paras os ciclos no estado corec a aplicação exibe apenas o comando Perfil de risco.
  • Quando a tela é aciona através de opção de menu, apresentar o comando Voltar.

Cenários de teste para E5.11. Consulta ciclo ativo da ES perfil supervisor.

Observação: agrupar a maior quantidade de itens de teste no mesmo teste.

Dado que:

  • registros de ciclos para ESs sob a responsabilidade do supervisor nos estados 1 - Em andamento; 2 - Corec; 3 - Pós - corec; 4 - Encerrado; 5 - Excluído.
  • registros de ciclos para ESs sob a responsabilidade de outro supervisor nos estados 1 - Em andamento; 2 - Corec; 3 - Pós - corec; 4 - Encerrado; 5 - Excluído.
Negócio:
  • Usuário logado não possui ES sob sua responsabilidade. Exibir a mensagem:
    • Nenhum registro que satisfaça os critérios de pesquisa foi encontrado. Refaça sua pesquisa alterando os critérios.
