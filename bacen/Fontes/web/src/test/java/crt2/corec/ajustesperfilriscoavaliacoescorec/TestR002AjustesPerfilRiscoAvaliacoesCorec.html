Cenários de teste para a estória 'Ajustes no perfil de risco para destacar avaliações Corec'

Histórico

Versão Data Responsável Descrição
1.0 24/03/2015 Christiano Nogueira Ferreira Versão Inicial
1.1 22/04/2015 David Nascimento Segunda versão
1.2 28/08/2015 David Nascimento Alterando os cenários CT004 e CT005 e incluindo os cenários CT006, CT007, CT008 e CT009 (a qual foram reordenados para CT010, CT011, CT012 e CT013) a fim de contemplar a mudança a qual o supervisor e o gerente ao detalhar ARC com ciclo em 'COREC', deve ser exibido sempre o ARC vigente.

Regras aplicáveis

As regras da estória encontram-se aqui.

Premissas iniciais

  • A lista de servidores foi carregada.

Cenários
