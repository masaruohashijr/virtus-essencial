Ligando e logando no sistema

  • Existem usuários com os perfis prontos para uso no sistema.
  • O sistema está em funcionamento.
    ligar o sistema SRC
  • O usuário efetua login.
    pausa a página de login
    a página contém Login do Sistema APS-SRC
    a página contém Acesso Usuário SISBACEN CPF
    clica na opção Usuário SISBACEN
    preenche unidade com #{unidade}
    preenche operador com #{usuário}
    preenche senha com #{senha}
    clica no botão Entrar
    a página contém o usuário #{unidade}_#{usuário}
    a página contém a data atual
