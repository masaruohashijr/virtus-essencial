Cenários

Cenários para validar que campos exibir para o sub-grupo 'Análise do supervisor para o elemento '<<Nome do elemento>>' .
Cenário
[C001] - Verificar a seção para ANEF cujo rascunho encontra-se no status: 'Previsto'.
Dado que
  • Os dados para este cenário encontram-se aqui.
Ação pretendida
  • Logar como: deliq.sisliq101
  • Acessar o menu: inicio/painelsupervisor
  • Clicar na ES: GOLDMAN SACHS
  • Clicar no ANEF: Solvência Patrimonial
Resultado esperado
  • O código da tela deve ser: [APSFW0212]
  • A seção Análise do supervisor para o elemento "Suficiência" não é exibida
  • A seção Análise do supervisor para o elemento "Qualidade" não é exibida
